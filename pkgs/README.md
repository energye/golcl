* bass  
[bass](http://www.un4seen.com/)库的Go接口。  

* libname  
自定义加载指定位置的libvcl或者liblcl。  

* libvlc  
libvlc库的Go接口。  

* macapp  
调试或者打包一个macOS下的app。  

* miniblink  
miniblink浏览器的Go接口。  

* skinh  
skinh皮肤库的Go接口。  

* winappres  
默认的windows下app资源。  

* wke  
wke浏览器的Go接口。  

* wintaskbar  
Windows7+开始支持的任务栏新特性。
